About This Project

In this project, I worked with a dataset of student results and used Python to:

Clean the data
Analyze student performance
Create charts and graphs

The goal is to make the data easy to understand and find useful insights.

Tools and Technologies

I used the following tools in this project:

Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

Dataset

The dataset used is:

student result.csv

It contains student marks and other related information which I have used for analysis.

What I Did
Cleaned the data
Checked important values
Compared student performance
Created graphs for better understanding
Output

In this project  you will see:

Graphs
Charts
Performance comparisons

It helps to easily understand how students are performing.
